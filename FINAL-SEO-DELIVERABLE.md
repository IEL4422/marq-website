## Implementation Code Snippets

### 1. How SEO Configuration is Applied

**File:** `src/utils/seo.ts`

The site uses a centralized SEO utility that updates all meta tags dynamically:

```typescript
export const updatePageSEO = (config: SEOConfig) => {
  // Updates document title
  document.title = config.title;

  // Sets meta description
  updateMetaTag('name', 'description', config.description);

  // Sets canonical URL
  if (config.canonical) {
    updateLinkTag('canonical', config.canonical);
  }

  // Sets Open Graph tags
  updateMetaTag('property', 'og:title', config.title);
  updateMetaTag('property', 'og:description', config.description);

  // Sets Twitter Card tags
  updateMetaTag('name', 'twitter:title', config.title);
  updateMetaTag('name', 'twitter:description', config.description);
};
```

### 2. Adding a New Page to SEO Configuration

**Step 1:** Add to `pageSEO` object in `src/utils/seo.ts`:

```typescript
export const pageSEO: Record<string, SEOConfig> = {
  // ... existing pages ...

  'new-page': {
    title: 'Page Title | 50-60 Characters | Marq',
    description: 'Meta description between 155-160 characters that describes the page content and includes a call to action.',
    canonical: 'https://marqtrademarks.com/new-page',
    keywords: 'primary keyword, secondary keyword, tertiary keyword',
    author: 'Marq'
  }
};
```

**Step 2:** Add route mapping in `src/App.tsx`:

```typescript
const pathToPageMap: Record<string, string> = {

```

**Step 3:** Use SEO hook in your page component:

```typescript

import { updatePageSEO, pageSEO } from '../utils/seo';

export default function NewPage() {
  useEffect(() => {

  }, []);

  return (
    <div>

      {/* Page content */}
    </div>
  );
}
```
